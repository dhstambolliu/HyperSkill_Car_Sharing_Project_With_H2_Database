Calling an instance method

You're given a method with the following declaration: public int getAddress(). There is also an already created instance john of the same class where this method is located.

How can you call this method? Write the invocation of the method in one line.

The given method is an instance method. Remember or check in the theory on how to call such methods. You only need to invoke this method for the john object. You do not need any more detail.
